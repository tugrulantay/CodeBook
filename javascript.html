JavaScript Documentation

1. What is JavaScript?

JavaScript makes websites interactive. If HTML builds the structure and CSS makes it look good, JavaScript makes it move and respond to users.

2. Variables

Variables store information. Think of them like boxes that hold data.

// Creating a variable
let name = "Jason";
let age = 25;

console.log(name);

3. Functions

Functions are blocks of code that run when you call them.

function sayHello() {
  console.log("Hello World!");
}

sayHello(); // Runs the function

4. If Statements

If statements let your program make decisions.

let age = 18;

if (age >= 18) {
  console.log("You are an adult");
} else {
  console.log("You are under 18");
}

5. Loops

Loops repeat code multiple times.

for (let i = 0; i < 5; i++) {
  console.log("Number: " + i);
}

6. The DOM

The DOM allows JavaScript to change HTML content.

document.getElementById("demo").innerText = "Changed!";

7. Events

Events happen when users click, type, or move the mouse.

document.querySelector("button").onclick = function() {
  alert("Button clicked!");
};

8. Arrays

Arrays store multiple values in one variable.

let fruits = ["Apple", "Banana", "Orange"];

console.log(fruits[0]); // Apple

9. Simple Example

This example changes text when a button is clicked.

function changeText() {
  document.getElementById("demo").innerText = "Hello JavaScript!";
}
