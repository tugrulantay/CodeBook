Python Documentation

1. What is Python?

Python is a programming language that is easy to read and simple to learn. It is used for websites, data science, automation, artificial intelligence, and more.

2. Variables

Variables store information. Think of them like labeled boxes that hold data.

# Creating variables
name = "Jason"
age = 25

print(name)

3. Functions

Functions are blocks of code that run when you call them.

def say_hello():
    print("Hello World!")

say_hello()  # Runs the function

4. If Statements

If statements allow your program to make decisions.

age = 18

if age >= 18:
    print("You are an adult")
else:
    print("You are under 18")

5. Loops

Loops repeat code multiple times.

for i in range(5):
    print("Number:", i)

6. Lists

Lists store multiple values in one variable.

fruits = ["Apple", "Banana", "Orange"]

print(fruits[0])  # Apple

7. User Input

You can ask the user to type something into the program.

name = input("What is your name? ")
print("Hello", name)

8. Modules

Modules are extra tools you can import into your program.

import math

print(math.sqrt(16))  # 4.0

9. Simple Example

This example checks if a number is even or odd.

number = 10

if number % 2 == 0:
    print("Even number")
else:
    print("Odd number")
