CSS Documentation

1. What is CSS?

CSS stands for Cascading Style Sheets. It is used to make websites look beautiful. If HTML builds the structure of a house, CSS paints the walls and arranges the furniture.

2. How CSS Selects Elements

CSS needs to know WHAT to style. We use selectors to choose elements.

/* This styles all paragraphs */
p {
  color: blue;
}

/* This styles elements with class="box" */
.box {
  background-color: lightgray;
}

/* This styles element with id="main" */
#main {
  font-size: 20px;
}

3. The Box Model (Very Important)

Every element on a webpage is like a box. That box has content, padding (inside space), border, and margin (outside space).

.box {
  width: 200px;       /* size of content */
  padding: 20px;      /* space inside */
  border: 2px solid black;
  margin: 10px;       /* space outside */
}

4. Flexbox (Easy Alignment)

Flexbox helps you align items in a row or column. It makes centering things very easy.

.container {
  display: flex;
  justify-content: center;  /* center horizontally */
  align-items: center;      /* center vertically */
}

5. CSS Grid (Rows and Columns)

Grid helps you create layouts with rows and columns, like a table but more powerful.

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

6. Positioning Elements

Position controls where elements appear on the screen.

.box {
  position: absolute;
  top: 50px;
  left: 100px;
}

7. Styling Text

CSS allows you to change fonts, size, weight, and spacing of text.

p {
  font-size: 18px;
  font-family: Arial, sans-serif;
  font-weight: bold;
}

8. Hover Effects & Animations

You can make buttons move or change color when users hover over them.

button {
  transition: 0.3s;
}

button:hover {
  background-color: gray;
  transform: scale(1.1);
}

9. Responsive Design

Responsive design means your website works on phones, tablets, and computers.

@media (max-width: 768px) {
  body {
    background-color: lightblue;
  }
}
